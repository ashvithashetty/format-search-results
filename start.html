Python web search

Returns the search results obtained from search engine in a clean format that can be easily used for further processing

search function takes the following parameters:

  • engine: search engine to be used, google or bing (string)
  • query: search string (string)
  • num: the number of results to be retrieved, cannot exceed 40 at a time (integer)
  • start: the number of top results to be ignored (integer)
  • recent: hour,day,week and month values are allowed
  • country code: 2 characters - valid country code (string of length 2)
